# 开始使用
请先[安装](#/doc/install)本组件库。

然后在你的项目中按需引入组件，代码如下：

```
import {Button, Tabs} from "monkee-ui"
```
就可以开始使用开关、按钮组件了。使用组件代码示例：

```html
<template>
  <div>
    <Button>按钮</Button>
  </div>
</template>
<script>
import {Button, Tabs} from "monkee-ui"
export default {
  components: {Button}
}
</script>
```
